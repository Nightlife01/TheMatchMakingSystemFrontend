About

Match a Maker is een applicatie ontworpen door 4 IT-studenten tijdens de Grasshopper Application Challenge. De App is ontwikkeld in Angular met een .NET API

De bedoeling van de App is het 'matchen' van makers en bedrijven. Bedrijven kunnen hier opdrachten posten waar makers zichzelf voor kunnen inschrijven.
